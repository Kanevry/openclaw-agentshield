## Regeln

1. **Jeder Fakt hat eine Quelle.** Keine Quelle = kein Fakt.
2. **Datum angeben.** Fakten veralten. "Stand: 27.03.2026" bei jedem Eintrag.
3. **Korrekturen sofort.** Wenn ein Fakt sich als falsch herausstellt, sofort korrigieren.
4. **Keine Meinungen.** SSOT enthält nur verifizierbare Aussagen.
